Living Painting Studio

Describe a scene in plain language. AI paints it. Then the painting comes alive — fog drifts, creatures wander, light shifts with the time of day, seasons slowly turn. It never loops the same way twice. You're the author of a world, not just a prompt.

Not a video. Not a screensaver. A living canvas.

The core experience

A painting that breathes.

You write a scene; the studio generates a beautiful still from it. Then a simulation layer brings that still to life — ambient particles, weather, autonomous creatures, and lighting that tracks the time of day and the turning seasons. The result is calm and never static: a world you set in motion and then simply live alongside.

You own the world, not the prompt. Tag elements, toggle their behavior, set the season, speed, and weather in real time. The canvas is yours to tune and keep — a piece that keeps painting itself.

Under the canvas

A still image, then a living simulation on top.

Generate

Your prompt becomes a base image via a hosted image model — the starting frame of the world.

Particle layer

Creature layer

Simple autonomous creatures that wander the scene on their own, never the same path twice.

Lighting layer

Time-of-day and seasonal light shifts, so noon, dusk, and winter all read differently.
